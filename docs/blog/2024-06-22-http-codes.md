## 2xx Successful

| Status Code | Meaning      |
|-------------|--------------|
| 200         | OK           |
| 201         | Created      |
| 204         | No Content   |

## 3xx Redirection

| Status Code | Meaning              |
|-------------|----------------------|
| 301         | Moved Permanently    |
| 302         | Found                |
| 304         | Not Modified         |

## 4xx Client Error

| Status Code | Meaning              |
|-------------|----------------------|
| 400         | Bad Request          |
| 401         | Unauthorized         |
| 403         | Forbidden            |
| 404         | Not Found            |
| 405         | Method Not Allowed   |
| 408         | Request Timeout      |
| 409         | Conflict             |
| 410         | Gone                 |
| 422         | Unprocessable Content|

## 5xx Server Error

| Status Code | Meaning                |
|-------------|------------------------|
| 500         | Internal Server Error  |
| 502         | Bad Gateway            |
| 503         | Service Unavailable    |
| 504         | Gateway Timeout        |
